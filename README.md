# site-leisu-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容相互独立，不针对任何特定域名或网站，仅作为静态页面集合进行存储与展示。仓库结构清晰，便于管理和引用。

## 目录说明

```
site-leisu-pages/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：每个独立页面以单独文件或子目录形式存放，命名遵循语义化规则。
- `assets/`：存放页面间共享的资源，如样式表、脚本和媒体文件。
- `index.html`：可作为页面导航或索引，非必需。

## 页面归档说明

- 所有页面均为纯静态 HTML，不依赖后端服务。
- 页面间无强关联，可独立部署或引用。
- 归档内容保持原始状态，仅做必要格式整理。
- 如需新增页面，请遵循现有目录结构，并确保资源引用路径正确。

## 维护说明

- 本仓库由维护者不定期更新，不承诺固定更新频率。
- 提交新页面或修改现有页面时，请保持代码整洁，避免引入外部依赖。
- 如发现页面显示异常或链接失效，可提交 Issue 反馈。
- 不建议对本仓库内容进行商业性使用或大规模转载。

## 许可

本仓库内容仅供归档与展示，不附加特定开源许可证。如需引用或使用，请自行评估风险。
